About

JuniorScript is a compiler for PHP, that makes everyday PHP development easier.

JuniorScript Installation

Recommended: Install from Terminal

curl -fsSLO https://raw.githubusercontent.com/JuniorCode/JuniorScript/master/juniorscript.sh && ./juniorscript.sh install

Copy and paste this command into your Terminal. This will automatically install the latest JuniorScript compiler.


Install from GitHub

./juniorscript install

Download JuniorScript from GitHub and type this command in your Terminal.

Examples

PHP Code:

$color = "red";
echo "My car is " . $color . ". ";
echo "My house is " . $color . ". ";
echo "My boat is " . $color . ".";

JuniorScript Code:

color = "red";
out join("My car is ", $color, ". ");
out join("My house is ", $color, ". ");
out join("My boat is ", $color, ".");

Output:

My car is red. My house is red. My boat is red.
